# The Maker Grid Macropad 🚀

A custom 9-key macropad featuring a rotary encoder and an OLED display, powered by the Seeed Studio XIAO RP2040. Designed for the Hack Club Hackpad YSWS.

## Bill of Materials (BOM)
To bring this to life, I need:
* 1x Seeed Studio XIAO RP2040
* 9x MX-style Switches (Designed around Hogback wheels/axles principles)
* 9x Blank White DSA Keycaps
* 9x 1N4148 Diodes (Through-hole)
* 1x 0.91" I2C OLED Display (SSD1306)
* 1x EC11 Rotary Encoder + Matching Knob
* M3 Heat-set inserts and M3 socket cap screws
